# dmarc-inspector

> Parse DMARC aggregate XML reports into readable tables. Supports gzip, files, and URLs.

## Overview

Parse DMARC aggregate XML reports into readable tables. Supports gzip, files, and URLs.

## Installation

### Binary (Linux x86_64)

```bash
curl -LO https://labs.stech-sol.com/downloads/dmarc-inspector/0.1.0/dmarc-inspector_linux_amd64
chmod +x dmarc-inspector_linux_amd64
sudo mv dmarc-inspector_linux_amd64 /usr/local/bin/dmarc-inspector
```

### From Source

```bash
git clone https://github.com/s-tech-labs/dmarc-inspector.git
cd dmarc-inspector
# Follow language-specific build instructions
```

## Quick Start

```bash
dmarc-inspector --help
```
